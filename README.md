# PixelPerfect Pro 🖼️

Redimensionador de imagens de alta performance que utiliza algoritmos de interpolação via HTML5 Canvas para garantir a melhor qualidade possível em aumentos (upscaling) e reduções de resolução.

**Processamento 100% Local**: Suas imagens nunca saem do seu navegador. Privacidade total e velocidade máxima.

## ✨ Funcionalidades

- **Resampling de Alta Qualidade**: Utiliza o motor do Canvas com suavização de imagem avançada.
- **Presets Inteligentes**: Formatos prontos para Instagram (Square/Story), 4K, 1080p e mais.
- **Controle de Proporção**: Trava de aspecto para evitar distorções.
- **Multi-Formato**: Suporte para PNG (Lossless), JPEG e WEBP.
- **Interface Moderna**: Design em modo escuro (Dark Mode) com animações suaves e suporte a Drag-and-Drop.
- **Privacidade**: Sem APIs de nuvem, sem rastreamento, sem IA externa.

## 🚀 Tecnologias Utilizadas

- **React 19** + **TypeScript**
- **Vite** (Build System)
- **Tailwind CSS 4.0** (Styling)
- **Lucide React** (Icons)
- **Motion** (Animations)

## 🛠️ Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para gerar a versão de produção:
   ```bash
   npm run build
   ```
